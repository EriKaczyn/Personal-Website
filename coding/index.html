"Big Dogs Tower Defense"

In my Objects and Design class, my classmates and I used Java to create this tower defense game. In the game, big dogs are trying to steal a small dog's food, and the player has to use towers to shoo the big dogs away. This project was a great experience because I had never worked on a project of this size before and it was a great way to practice my JavaFX skills.
